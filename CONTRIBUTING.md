# Contributing

Thank you for looking in this file!  There are different ways of contributing to VideoBot, and we appreciate all of them.


## Reporting Bugs

Please report bugs at https://github.com/TIC4302/video-streaming-bot/issues and give label as `bug`.


## Feature Requests

Please submit your feature requests at https://github.com/TIC4302/video-streaming-bot/issues/<number>.
The <number> will be updated for every release.

## Code of Conduct

We follow the [Apache Software Foundation Code of Conduct](https://www.apache.org/foundation/policies/conduct).
